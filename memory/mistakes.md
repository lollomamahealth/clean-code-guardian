# Mistake Log

Errors encountered during development, categorized for learning.

---

## Categories

- **PATTERN**: Code pattern anti-patterns (Pydantic, async, SQL)
- **TEST**: Test-related failures
- **RUNTIME**: Runtime errors from executed code
- **LOGIC**: Logical errors in implementation
- **DUPLICATE**: Code that duplicated existing utilities

---

## Log Format

```
### [CATEGORY] Short description
- **Date**: YYYY-MM-DD
- **File**: path/to/file.py
- **Error**: Brief error message
- **Mistake**: What was wrong
- **Correction**: How it was fixed
- **Prevention**: Pattern to block in future
```
